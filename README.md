# Boomaga-IPP
Boomaga-IPP (BOOklet MAnager - IPP) is intended to be a modern (as of 2026) reimplementation of the venerable <a href="https://www.boomaga.org/">Boomaga virtual printer system</a> (unmaintained since 2019). Boomaga-IPP is written entirely in Rust and provides print preview and booklet printing capabilities using the IPP Everywhere protocol, rather than the original Boomaga's CUPS backend drivers. It is designed specifically for Linux systems running Wayland graphical environments and managed by systemd. The Boomaga-IPP system comprises an IPP Everywhere service and a Rust-based GUI application that enables users to preview documents before printing and create various print layouts, including booklets and multi-page arrangements.
